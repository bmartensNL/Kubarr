# Kubarr API Documentation

*Documentation in progress*

This document will provide REST API reference for integrations with Kubarr.

## API Endpoints

The Kubarr backend exposes a REST API on port 8000.

### Health Check

```
GET /api/health
GET /api/system/health
```

Returns the health status of the backend service.
